Submit Your Data

We realize that we haven't listed every Philadelphia-based open data set in the OpenDataPhilly catalog. If you have a specific data set that you think should be included, let us know by completing and submitting the form below. We'll review the information and determine whether the data set should be included in OpenDataPhilly. The data must be specific to Philadelphia, available online, and contain structured data. A full list of review criteria is available on the FAQ page.

This form is for recommending specific data sets, applications, or APIs. If you'd like to nominate a general area where you think data should be more easily available, visit the Nominate Data page.
